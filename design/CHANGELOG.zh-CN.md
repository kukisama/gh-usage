# 更新日志

[English](CHANGELOG.md) | 简体中文

这里记录 **gh-usage** 每个版本的重要变化，站在你的角度讲清楚：有什么新功能、哪里变好了、对你有什么用。

本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

---

## [1.2.0] — 2026-06-14

把报告打磨得能放心分享给任何人。

### 新增

- **一键整页截图。** 报告里新增了相机按钮，点一下就能把整页存成一张小巧的图片（约 430 KB）——发群、贴文档、发进度，都不用再折腾截图工具。
- **一键隐私遮蔽。** 新增隐私开关，点一下就能把主机名、项目名、会话标题全部打码，分享时不怕泄露敏感信息。截图时还会自动开启，再也不怕手滑泄密。

### 优化

- **体积更小，更新更快。** 程序大幅瘦身（Windows 版本约小了 45%），装起来、升级起来都更轻快。
- **截图更清晰、更靠谱。** 截图改用浏览器自身来渲染，打码字段的样子和你在屏幕上看到的一模一样。

## [1.1.0] — 2026-06-05

让冰冷的数字，变成一份你真正愿意打开的报告。

### 新增

- **自包含 HTML 报告。** 每次运行都会生成一个精美的单文件 HTML，任意浏览器双击即看——不用服务器、不用数据库、不用联网。里面有核心指标、每日用量图表、模型分布，还有能搜索、能排序、能翻页的明细记录表。
- **多语言界面。** 报告内置中英文，在页面上一键切换。
- **多机合并。** 在多台电脑上各跑一次 `gh-usage`，把 CSV 收进一个文件夹，用 `--merge` 就能合并成一份带「按机器对比」的汇总报告——团队复盘、换设备都很顺手。

### 优化

- 记录提取更准确，并修复了若干稳定性问题。

---

## [1.0.0] — 2026-05-18

第一个稳定版本——快、纯本地、直击要点。

### 新增

- **本地用量扫描。** 让 `gh-usage` 扫一遍你的电脑，它就会从本机已有的文件里整理出 GitHub Copilot 使用记录——全程不上传。
- **导出即可进表格。** 结果能导出成 CSV（默认带 UTF-8 BOM，Excel 打开不乱码），也能导出 JSON 接自动化。
- **终端即时摘要。** 每次运行都会在终端打印一段紧凑摘要——总 credits、活跃天数、记录条数、日均用量，一眼看完。
- **Windows 装起来很省事。** 通过 Windows 包管理器一行搞定：`winget install gh-usage`。
